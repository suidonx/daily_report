## わかったこと
### CSS
- transform
	- translate(X, Y)：X軸、Y軸方向に移動
	- translateX(数値)：X軸のみ移動
	- translateY(数値)：Y軸のみ移動
レイアウトを崩さずに要素を動かすことができる。
動かしたからといって見え方が変わるだけで要素自体は動いていないことに注意
要素も動かしたいならposition: relative、position: absolute;を使う。

### JavaScript
**Web API**
https://jsonplaceholder.typicode.com/users

- Web APIとはWeb上でデータや機能を提供する仕組み
- REST APIはRESTの設計思想に乗っ取り作られたAPI

- 全ユーザーの情報、特定ユーザーの情報といったデータのリソースをURIで表す。

**REST API**

| リソース   | URI例        |     |
| ------ | ----------- | --- |
| 全ユーザー  | `/users`    |     |
| 特定ユーザー | `/users/1`  |     |
| 投稿     | `/posts`    |     |
| 特定投稿   | `/posts/10` |     |

上記の例でだと/usersで10人のユーザーのデータをJSON形式で取得できる。

ドメイン名がAPIサーバーとなる。

GETやPOSTといったHTTPメソッドでリソースを操作する。
データを取得したり、データを編集したり、データを作成したり・・・

実際にJavaScriptでWeb APIを叩くときは
windowオブジェクトのfetch()メソッドを使う
promiseオブジェクトが返ってくる。

responseオブジェクトを受け取るには
async function() {}
非同期関数にして、awaitを使って書いていく。

### CSS
**aタグを親要素いっぱいに貼る**
親要素はposition: relativeにして
子要素を以下のように記述すると、親要素いっぱいになる
```css
position:absolute;
  top:0;
  left:0;
  width:100%;
  height:100%;
```

**一部だけaタグのリンクを上書きしたい時**
その要素に対して、position: relative;を記述
そして、z-indexの値を上書きしたいやつよりも大きい値にする。
ちなみに、初期値はautoになっていて、position指定されていないとz-indexは使えない。

**aaa...とtextareaに入力するとコンテナの幅を超えてしまう問題**
```
word-break: break-all;
// 強め
// はみ出しそうになったら強制的に折り返す。
overflow-wrap: break-word;
// 弱め
// 長い単語で折り返す。折り返すときに句読点がこないようにするといった配慮がされる。aaa...ははみ出てしまう。

```
メリット、デメリットは下記を参照。
https://qiita.com/ist-a-k/items/2b1385574e1a1babdde1#word-wrap-break-word

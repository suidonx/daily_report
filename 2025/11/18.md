## わかったこと
### Django
- querysetに対してlist()とするとlistにできる。
- listにすることでモデルオブジェクトがバラバラでも一つにまとめることができ、さらにソートもできる！
- 普通にやると、いいね、リツイート、コメントをそれぞれ降順にすることはできるが、全てまとめて降順に並べるというのは上手くやらないとできない。
```
notifies = sorted(
	list(new_like) + list(new_retweet) + list(new_comment),
	key=lambda x: x.created_at,
	reverse=True,
)
```

## わかったこと
- ビルド
	- Reactを使った書いたJSX, TSXなどをJS/CSS/HTMLといったブラウザが読めるファイル形式に変換する必要がある
	- Viteなどのビルドツールではさらに
		- ファイル結合、ファイル圧縮を行う
	- npm run buildしたときにViteではビルドした結果をdistディレクトリに保存する
- GitHub Actions
	- buildした結果をdistディレクトリに格納、deployを一通り行うために必要なワークフロー
	- GItHub側でビルドする
- useCallback
	- `const fn = useCallback(() => {}, []);`
	- 依存配列が変わらない場合、前の関数を返す
	- 変わった場合、関数を再生成
	- アロー関数はレンダリングしたときに関数を生成するので、無駄に生成するのを防ぐ
- memo
	- propsが変わらない限り再レンダリングしない
- リファクタリング方針
	- 責務で分割
	- ロジックとUIを分けていく
	- イベントハンドラの命名
		- https://zenn.dev/counterworks/articles/event-handler-basic-rule

- 型定義
	- (引数名: 型) => 戻り値の型
